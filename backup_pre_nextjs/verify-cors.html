Firebase Storage CORS Verification

How to Configure CORS (Google Cloud Console Method)

  1. Go to Google Cloud Console - Storage
  2. Select your project: ro7mbms
  3. Click on your bucket: ro7mbms.firebasestorage.app or ro7mbms.appspot.com
  4. Click the Configuration tab
  5. Scroll to CORS configuration
  6. Click Edit CORS configuration
  7. Paste this JSON: 
    [
  {
    "origin": [
      "https://r7communications.vercel.app",
      "http://localhost:5500",
      "http://127.0.0.1:5500",
      "http://localhost:3000"
    ],
    "method": ["GET", "HEAD", "OPTIONS"],
    "responseHeader": ["Content-Type", "Content-Length", "Content-Disposition"],
    "maxAgeSeconds": 3600
  }
]
  8. Click Save
  9. Wait 2-5 minutes for changes to propagate
  10. Test again using the button above
